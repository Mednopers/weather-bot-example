Weather bot example
===================

Simple weather bot example

Software
--------
You need PHP, GIT, Composer, Postman to run the application.

Installation
------------
Download and install application
using Git:
```bash
$ git clone https://github.com/Mednopers/weather-bot-example.git
$ cd weather-bot-example
$ composer install
$ php -S 127.0.0.1:8080
```
After executing these commands, the necessary components will be installed and configured.
The web server start listening.

Usage
-----
There is no need to configure a virtual host in your web server to access the application.

Now you can send GET or POST requests to the application via your Postman at <http://127.0.0.1:8080//weather-bot>.
Request example available in the postman directory.
